# بوت ألعاب ديسكورد مع عجلة روليت متحركة

## طريقة التشغيل

1. ضع توكن البوت الخاص بك في ملف `.env` بهذا الشكل:
   ```
   BOT_TOKEN=هنا_التوكن
   ```
2. ثبّت المكتبات:
   ```
   npm install
   ```
3. شغّل البوت:
   ```
   npm start
   ```
4. جرّب أمر: `!روليت`  
   وسيتم عرض عجلة متحركة بأسماء اللاعبين واختيار الفائز تلقائياً.

## المتطلبات
- Node.js v16 أو أحدث
- صلاحية قراءة أعضاء السيرفر وكتابة الرسائل

---

إذا احتجت إضافة ألعاب أو أوامر أخرى، تواصل معي وأضف طلبك!
